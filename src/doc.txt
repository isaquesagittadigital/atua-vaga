1. TESTE COMPORTAMENTAL BIG FIVE
========================================================

Backend psicométrico implementado com 30 facetas de personalidade.

Arquivos criados/modificados:
- server/routes/behavioral-test.ts  → API para calcular resultados Big Five
- src/components/pages/candidate/BehavioralTestPage.tsx → Página do teste
- src/components/pages/candidate/TestSuccessModal.tsx → Modal de sucesso

Tabelas Supabase:
- behavioral_tests → Testes cadastrados
- test_questions → 30 perguntas Big Five (seed)
- candidate_test_results → Respostas e scores dos candidatos

Fluxo:
1. Candidato acessa Dashboard → clica "Fazer teste comportamental"
2. Sistema busca o ID do teste Big Five automaticamente
3. Candidato responde 30 questões em escala Likert
4. Progresso é salvo em tempo real no Supabase
5. Ao finalizar, backend processa as 30 fórmulas matemáticas
6. Modal de sucesso exibe o resultado
